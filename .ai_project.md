## Tech Stack

- Next.js 16 / React 19 / TypeScript
- Vitest 4 / Hardhat 3 / Solidity 0.8.36
- iExec Nox handle SDK / Nox protocol contracts / Ethereum Sepolia
- Viem calldata 编码；KeeperHub Direct Execution 仅用于安全模拟，MCP 为比赛期真实执行层

## Key Commands

- `npm run dev`：启动本地工作台
- `npm run typecheck && npm run lint`：静态检查
- `npm test`：TypeScript + Solidity 测试
- `npm run test:e2e`：真实 Chrome 端到端与三视口验收
- `npm run build`：生产构建
- `npm run keeperhub:preflight`：离线检查 KeeperHub/Sepolia 前置条件，不回显配置值
- `npm run keeperhub:handoff -- --opportunity-id <id>`：离线生成 simulate-only 合约调用交接包
